# Boundary Run: The Little Signal — Game Specification

Version: 8.8.3

Boundary Run is an AxonOS Education browser game about protecting cognitive privacy through emotional play.

The player protects Ari, a small safe-intent courier. Ari carries a choice, not a thought. Kibo, a loyal guardian dog, helps warn and protect Ari.

Core gameplay: see threat, protect Ari, learn why, receive replay proof.

Primary systems:

- Three lanes: Raw Stream, Boundary Layer, Safe Intent Path.
- Actions: Audit, Revoke, Throttle, Seal Vault, Quarantine.
- Threats: Raw Signal Leak, Stale Consent, Unauthorized App, Artifact Spike, Unsafe Stimulation, Memory Phisher.
- Metrics: Boundary Integrity, Privacy Leakage, Consent Coherence, Latency Pressure, Ari Trust.
- Collectibles: Proof Shards, Consent Tokens, Vault Keys.
- Replay proof: deterministic hash generated from seed, inputs, and game state transitions.

Release rule: this is an educational simulation, not a medical device, and it collects no real neural data.
