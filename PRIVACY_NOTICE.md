# Privacy Notice — Neural Boundary Game v7.3.0

**Data minimisation by architecture. No accounts. No telemetry. Local-only.**

## What this game does not collect

Neural Boundary Game v7.3.0 (community edition) does not intentionally collect:

- Account data or email addresses
- Real names or location data
- Advertising identifiers
- Gameplay telemetry or analytics
- Raw neural, biometric or health data
- Wallet addresses, private keys or seed phrases
- Session-replay recordings
- Cross-site tracking data

## Local storage only

The game stores the following in your browser's local storage under the
`axonos_nbg_v5512_` namespace: selected mode and difficulty, best scores per mode,
daily seed progress, and the last replay (optional export). Nothing is transmitted
to any server during gameplay.

## Hosting infrastructure

The game is hosted on GitHub Pages. GitHub may log standard HTTP metadata (IP
address, user agent, referrer) as part of their normal hosting operations.
AxonOS does not receive or process those logs. See GitHub's privacy policy for
details.

## Reset

You can remove all `axonos_nbg_v5512_*` local storage keys at any time using
the "Reset Local Data" option in the game.
